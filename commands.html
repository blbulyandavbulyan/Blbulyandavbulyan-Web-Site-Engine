Команды:

Общие команды для всех пользователей:

  • Команда SEND MESSAGES предназначена для отправки сообщения(собщений) пользователю (пользователям):
    • SEND MESSAGES "%message%" TO USERS %username%, %username2%, ...
      • "%message% - текст сообщения
      • %username% - имя пользователя, котрому будет отправлено сообщение
      • %username2% - второе имя пользователя котрому будет отправлено это же сообщение
      • ... - и так далее, через запятую перечислить всех пользователей которым нужно послать сообщение
    • SEND MESSAGE "%message%" TO USER %username%
      • "%message% - текст сообщения
      • %username% - имя пользователя, котрому будет отправлено сообщение
  • GET INFO ABOUT THE %username% - получить информацию о пользователе
    • %username% - имя пользователя
  • READ MESSAGES FROM USER %username% - считать сообщения от пользователя
    • %username% - имя пользователя
    READ MESSAGES FROM USERS %username%, %username2%, ...
    • %username% - первое имя пользователя от которого нужно считать сообщения
    • %username2% - второе имя пользователя от которого нужно считать сообщение
    • ... - и так далее перечислить через запятую пользователей от которых нужно прочесть сообщение
  • READ ALL MESSAGES - считать все сообщения
  • CHANGE VARIABLE %variable% FOR ME %VariableValue% - эта команда изменяет переменую пользователя (требует подтверждение самого пользователя)
    • %VariableValue% - значение переменной
    • %variable% - это переменная, тут можно использовать следующее:
      • EMAIL- это Email
      • PASSWORD - это пароль
      • UserFirstName - это имя
      • UserLastName - это фамилия

Команды для администраторов и суперадминистратора и модераторов

Команда WARNING - предназначена для предупреждения пользователей

Описание синтаксиса команды WARNING:

    WARNING %WarningType% %CauseOfWarning%
  • %WarningType% - тип предупреждения, может принимать следующие значения:
    • VIOLATION - нарушение, в таком случае в %CauseOfWarning% должен содержаться пункт(ы) правил(а), котор(ые, ое) пользователь нарушил, перечислять через запятую, если таких пунктов несколько
  • WARNING %username% %rule%
  • BAN USER %username% - заблокировать пользователя
  • BAN USERS %username%, %username2%, ... - заблокировать пользователей (имена пользователей перечислять через запятую)
  • UNBAN USER %username% - разблокировать пользователя
  • UNBAN USERS %username%, %username2%, ... - разблокировать пользователей (имена пользователей перечислять через запятую)

Команды только для суперадминистратора

    Команда SET VARIABLE:

  • Может принимать разные параметры, но смысл всегда один она что-то устанавливает или изменяет, (доступна только суперадмину) SET VARIABLE %variable% FOR %username% %value
    • %variable% - это переменная, тут можно использовать следующее:
      • EMAIL- это Email
      • PASSWORD - это пароль
      • UserFirstName - это имя
      • UserLastName - это фамилия
    • %username% - это имя пользователя для которого премениться команда
    • %value% - значение перемнной

    • Примеры использования:

    • SET VARIABLE PASSWORD FOR test 1234 - эта команда установит пароль 1234 для пользователя test

    Особености работы:

    • Суперадминистратор не может ей воспользоваться для себя, то есть свои данные суперадмин этой командой менять не может!
      Это сделано по причине того что эта команда меняет данные без участия владельца аккаунта,
      то есть от пользователя для которого применена эта команда, для измениния своих данных суперадмин должен воспользоваться командой CHANGE.
    • Эта команда не требует действий пользователя на которого она направлена

    Команда BROADCAST - предназначена для рассылки сообщений всем пользователям
